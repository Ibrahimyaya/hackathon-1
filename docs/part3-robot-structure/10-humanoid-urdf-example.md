# Building a Complete Humanoid URDF

> **Learning Goal**: Create a realistic humanoid robot URDF with arms, legs, torso, head, and realistic joint limits.

## Overview

A humanoid robot requires a complete URDF describing:

- **Torso**: Central body mass
- **Arms**: 2 arms with 3 joints each
- **Legs**: 2 legs with 3 joints each
- **Head**: 1 joint for pan
- **Sensors**: IMU, cameras
- **Joint limits**: Realistic ranges of motion
